# The Lash Book

This guide teaches Lash as a programming language for Bash automation.

Lash keeps shell scripts close to Bash, but adds a parser, semantic checks, structured blocks, arrays and maps, compile-time directives, and a small amount of safer syntax around common shell patterns.

Use this book for learning and examples. Use the [language spec](../language-spec.md) when you need the exact implemented surface.

## How to read this

1. Read chapters 1-6 first. They cover the Lash syntax you will write most often.
2. Use chapters 7-9 when you need shell interop, jobs, traps, or compile-time directives.
3. Keep chapter 11 nearby when porting Bash; it calls out the behavior that is easy to misread.
